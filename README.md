Application Web de Collecte de Données Académiques

Description

Ce projet est une application web développée en Python avec le framework Flask et une base de données MySQL. Elle permet de collecter, stocker et afficher des informations académiques sur des étudiants à travers une interface simple et intuitive.

L’utilisateur peut saisir les informations suivantes : matricule, nom, filière, niveau (L1, L2, L3) et note. Les données sont ensuite enregistrées dans une base de données MySQL et peuvent être consultées sous forme de tableau.

Technologies utilisées

Python 3
Flask
MySQL

Fonctionnalités

Ajout d’un étudiant
Sélection de la filière à partir d’une liste
Sélection du niveau (L1, L2, L3)
Enregistrement des notes
Affichage des données sous forme de tableau
Interface utilisateur simple et responsive
